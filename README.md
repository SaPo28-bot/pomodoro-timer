A clean, modern Pomodoro timer with a dark theme featuring 25-minute work sessions and 5-minute breaks.

## Features

- **25-minute work timer** with 5-minute breaks
- **Start/Pause/Reset controls**
- **Automatic mode switching** (work ‚Üí break ‚Üí work)
- **Progress bar** showing time remaining
- **Session tracking** and statistics
- **Responsive design** that works on all devices
- **Modern dark theme** with gradient accents
- **Visual feedback** when modes change

## Files

- `pomodoro-timer.html` - Full-featured version with advanced styling
- `pomodoro-simple.html` - Simplified, clean version perfect for quick use

## How to Use

1. Open either HTML file in your web browser
2. Click **Start** to begin the 25-minute work timer
3. The timer will automatically switch to a 5-minute break when work time ends
4. Use **Pause** to pause the timer, **Reset** to start over
5. Track your sessions and total focus time

## Customization

You can easily customize the timer durations by modifying the JavaScript variables:
- Work time: `25 * 60` (25 minutes in seconds)
- Break time: `5 * 60` (5 minutes in seconds)

## Technologies

- CSS3 (with modern features like CSS Grid, Flexbox, and custom properties)
- Vanilla JavaScript (no dependencies)
